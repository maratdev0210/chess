Board Component

Description:

The Board class represents the chess board, the position of pieces and methods related to constucing the board

List of properties:
  • The constuctor(position)
  • getPosition()
  • updatePosition(move)
  • displayBoard()
  • getPiece(square)

constructor(position):

Returns new object of type Board. The argument list passes the chess position which is represented as the following object structure:

position: [{color: 'white' || 'black', piece: 'pawn' || 'knight' || 'bishop' || 'king' || 'queen' || 'rook', position: algrebraic chess notatiion position}]

creates a matrix version of the board from the given position. The matrix representation of the board is used for finding all the legal moves.

Board.getPosition()

Returns the current position from the instance of Board class. The returned position represents the object position and matrix representation of the current position of instance Board.

Board.updatePosition(move)

Returns new Board class based on the given move*. The method is called if and only if the given move is legal.

The move object represents the following structure:

move: {piece: 'pawn' || 'knight' || 'bishop' || 'rook' || 'king' || 'queen', from: initial position of the piece, to: final position of the piece}

The algorithm for updating the position:

  1. Search for the given position given in from property of move object.
  2. Delete the piece standing on that position and add new object to position with the color of the piece, piece type and its new position
  3. Call the Board consturctor passing to its argument list a new position object

Board.displayBoard()

Displays the board on the browser.

Board.getPiece(square)

Returns the piece that is placed on the given square or null if the square is empty. This method is used for Legal Moves component

Game State component

Description

The gameState class represents the current state of the game, the turn, the checkmate state and other things

List of properties:
  • The constructor(turn)

constructor(turn)

Creates a new instance of game state class. The game State class stores the current move, the number of moves without capture or pawn movements, the stalemate state(true or false), the checkmate state(true or false).
